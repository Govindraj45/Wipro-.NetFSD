# Coding Challenge 39 - Task Management Security App

`TaskManagementSecurityApp` covers:

- secure forms-based authentication with cookies
- role-based access for `Admin` and `User`
- claims-based edit permission with `CanEditTask`
- anti-forgery protection on login, registration, logout, and task editing
- secure cookie settings and 15-minute session expiration
- Razor output encoding for task content

Seeded users:
- `admin` / `Admin@123`
- `member` / `Member@123`
