# Wipro .NET Full Stack Developer (FSD) Training

Welcome to the Wipro .NET Full Stack Developer training repository! This repository serves as a comprehensive portfolio and workspace containing daily hands-on assignments, mini-projects, and conceptual demonstrations built throughout the training program.

## 🚀 Overview

This repository is structured day-by-day, documenting the progressive learning journey from C# fundamentals to advanced .NET Core web development and database management. It includes practical implementations of design patterns, unit testing, and web applications.

## 📂 Repository Structure

The project is organized into folders corresponding to the daily topics and modules covered:

- **Day 1 to Day 7: C# Fundamentals & OOP**
  - Basic syntax, control flows, and Object-Oriented Programming principles.
  - Implementations of algorithms (Sorting: Bubble, Insertion, Selection, Merge, Heap; Searching: Linear, Binary).
  - Exception handling, arrays, and basic data structures.

- **Day 8: Advanced C# Concepts**
  - Delegates, Anonymous Methods, and Lambda Expressions.
  - Examples: *Student Activity Evaluation System*.

- **Day 10: Unit Testing**
  - Test-Driven Development (TDD) using xUnit.
  - Examples: *Calculator Unit Testing* project verifying core arithmetic logic and edge cases.

- **Day 11 to Day 17: Database & Data Access**
  - Relational database concepts, Normalization (Day 16-3), and complex SQL queries (Day 15-2 Subqueries).
  - ADO.NET and Entity Framework fundamentals.

- **Day 18: ASP.NET Core Web Development**
  - **Student Registration Portal:** Built using ASP.NET Core Razor Pages.
  - **Course Registration Portal:** Built using ASP.NET Core MVC with comprehensive Model validation, tag helpers, and routing.

- **SOLID Principles Demo (`Solid Demo PPT`)**
  - Interview-ready implementation of the five SOLID design principles (SRP, OCP, LSP, ISP, DIP) in C# with well-structured, robust code examples.

## 🛠 Technologies & Tools

- **Language:** C# 12.0
- **Framework:** .NET 8.0 (Console, MVC, Razor Pages)
- **Testing:** xUnit
- **Frontend:** HTML5, CSS3, Bootstrap 5 (for web apps)
- **IDE / Environment:** Visual Studio, VS Code, .NET CLI

## 🏃 Getting Started

To run any of the projects locally on your machine:

1. Clone this repository:
   ```bash
   git clone https://github.com/Govindraj45/Wipro-.NetFSD.git
   ```
2. Navigate to the specific day's directory (e.g., `cd Day18/CourseRegistrationPortal`).
3. Build the project to restore dependencies:
   ```bash
   dotnet build
   ```
4. Run the application:
   ```bash
   dotnet run
   ```
