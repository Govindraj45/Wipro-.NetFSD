# Day 5 - Advanced C# Concepts

This console application demonstrates C# 7 and C# 8 features, exception handling, file handling, delegates, events, reflection, attributes, SOLID principles, and design patterns.

## C# 7 and C# 8 Feature Table

| No. | Feature | Version | Description |
| --- | --- | --- | --- |
| 1 | Tuples | C# 7 | Return multiple values without creating a separate class. |
| 2 | Out variables | C# 7 | Declare out variables directly inside method calls. |
| 3 | Pattern matching | C# 7 | Check object shape and type safely using patterns. |
| 4 | Local functions | C# 7 | Keep helper logic close to the method that uses it. |
| 5 | Deconstruction | C# 7 | Break a tuple or object into separate variables. |
| 6 | Switch expressions | C# 8 | Write compact expression-based selection logic. |
| 7 | Nullable reference types | C# 8 | Identify possible null values earlier. |
| 8 | Using declarations | C# 8 | Dispose resources automatically with less nesting. |
| 9 | Indices and ranges | C# 8 | Access values from the end or slice collections easily. |
| 10 | Async streams | C# 8 | Consume asynchronous data using `await foreach`. |

## Other Topics Covered

| Topic | What the app demonstrates |
| --- | --- |
| Exception handling | `try`, `catch`, `finally`, and `FormatException` handling. |
| File handling | Creating, writing, reading, and parsing a text file. |
| Delegates | Passing discount logic as a callback. |
| Events | Raising a payment processed event. |
| Reflection | Inspecting class and method metadata at runtime. |
| Attributes | Creating and reading custom attributes. |
| SOLID | Table explaining all five SOLID principles. |
| Design patterns | Strategy, Factory, and Dependency Inversion examples. |
