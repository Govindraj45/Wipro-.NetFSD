# Day 5-2 - Pattern Matching With Switch Statement

This console application demonstrates pattern matching in C# using:

| Step | Concept | Description |
| --- | --- | --- |
| 1 | Input conversion | Reads user input and converts it into `int`, `double`, or `string`. |
| 2 | Type patterns with switch | Uses `case int`, `case double`, and `case string`. |
| 3 | Tuple patterns | Matches a tuple and extracts values. |
| 4 | Property patterns | Checks object properties such as `Name`, `Role`, and `NameLength`. |

Sample input used for verification:

```text
Govindraj
```
