# W5 Day 5 - Advanced Library Management System

This folder contains `AdvancedLibraryManagement`, an ASP.NET Core MVC assignment using EF Core, the Repository Pattern, and AJAX.

## Covered Requirements

- Generic repository interface with CRUD operations.
- EF Core generic repository implementation.
- Specific repositories for books, authors, and genres.
- One-to-many and many-to-many EF Core relationships.
- Asynchronous CRUD methods.
- Optimized book queries with eager loading, search, ordering, and pagination.
- AJAX-powered page for listing, adding, searching, and deleting books without a full page reload.
- AJAX endpoints for author and genre CRUD, plus page forms for adding authors and genres.
- Server-side and client-side error handling for AJAX operations.

Run the app and open `/Books` to use the AJAX interface.
