# W4 Day 5 - Advanced Routing MVC Application

`AdvancedRoutingStoreApp` covers the routing requirements from the PDF:

- complex MVC routes such as `/products/{category}/{id}` and `/users/{username}/orders/{orderId}`
- dynamic routing behavior for checkout and dashboard experiences
- custom route constraints for allowed categories, allowed price ranges, and GUID-based promo links
- unit tests for custom constraints and dynamic routing behavior
