# W5 Day 1 - Advanced MVC Filters

`AdvancedMvcFiltersApp` covers the filter requirements from the PDF:

- logging filter with DI-backed dependencies
- authentication filter for protected actions
- role authorization filter for admin-only flows
- global exception filter for user-friendly error handling
- separate store and banking controller scenarios to reflect both assignments in the PDF
- unit tests for authentication, authorization, and error filter behavior
